### Getting Started / Download

```
git clone https://github.com/3ndrius/portfolio.git
cd portfolio
npm install
```

## Run

```
npm start
```

### Develop

```
After install app, add your own bucket slug to app in
- App
- PartGrid
- Single
- Contact

```

### Run in build react mode

```
npm run build
```
